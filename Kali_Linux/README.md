VMware Fusion + Kali Purple Cybersecurity Lab

Project Overview

This project documents the deployment of a cybersecurity lab environment using VMware Fusion on Apple Silicon (M1 Max) and Kali Purple Linux.

The objective is to create a secure sandbox environment for cybersecurity training, penetration testing practice, blue team exercises, and networking labs without impacting the host operating system.

Environment

* Host Machine: MacBook Pro M1 Max
* RAM: 32 GB
* Virtualization Platform: VMware Fusion 26H1
* Guest Operating System: Kali Purple Linux ARM64
* Virtual CPUs: 4
* Memory Allocation: 8 GB
* Network Mode: NAT

Lab Goals

* Practice Linux administration
* Build cybersecurity skills
* Conduct penetration testing exercises
* Perform network analysis
* Create isolated environments for security research
* Develop blue team and defensive security knowledge

Installation Process

Step 1: Install VMware Fusion

Downloaded and installed VMware Fusion 26H1 for Apple Silicon.

Step 2: Download Kali Linux ARM64

Downloaded the ARM64 ISO image compatible with Apple Silicon devices.

Step 3: Create Virtual Machine

Configured:

* 4 CPU Cores
* 8 GB RAM
* 60 GB Virtual Disk
* NAT Networking

Step 4: Install Kali Purple

Installed Kali Linux using the graphical installer and selected:

* XFCE Desktop Environment
* Top 10 Security Tools
* Recommended Kali Tools
